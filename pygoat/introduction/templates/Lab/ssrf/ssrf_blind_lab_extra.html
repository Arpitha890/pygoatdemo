LAB EXPERT

This lab is the enhanced version of the fist lab for blind SSRF. This should illustrate the danger of SSRF when an application is not monolithic and uses a multi-service achitecture like those employed in cloud applications.

In order to illustrate this, pygoat must be run together with additional services.

First, run this project from a fedora, or centos OS. Then in this OS install Redis (version 5) and run it.

There is a way to get a reverse shell by using the same SSRF vulnerability from lab1 to get a reverse shell into the server running pygoat.

You can use the the installation script provided in this project named install_redis.sh, to install and run redis.
